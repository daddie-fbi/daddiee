# YOLOv8 Object Detection Lab

## Setup
pip install ultralytics opencv-python

## How to Run

### Image Test
python lab_1.py

### Video Test
Edit lab_1.py and change source to video file
python lab_1.py

### Webcam Test
Edit lab_1.py and change source to 0
python lab_1.py

## Files
- lab_1.py: Main script
- test_image.jpg: Test image
- test_video.mp4: Test video
